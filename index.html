Model - P181 Digiflow Indicator

NUCLUS model P181 is a flow rate indicator. This model can be mounted either panel or field mounting. The P181 model has programmable scale factor with direct flow rate reading in M3/hr / LPH / LPM / LPS / GPM us / GPM uk.

Technical Specifications:

  • Function: Flow Rate indicator
  • Display: Backlit alphanumeric 1 x 16 LCD
  • Supply Voltage: 230 V AC
  • Linearity: + 0.5 %
  • Accuracy: + 1 % of FSD
  • Input: from Nuclus sensor P812
  • Scale Factor: 00.1000000
  • Configuration Lock: Password Protection
  • Calibration: Using front membrane keys
  • Housing: Plastic ABS
  • Protection: IP 65 for field mounting
  • Meter Size: 110mm X 110 mm X 140mm
  • Cutout Size: 90 mm X 90 mm
  • Operating Temp.: 0 to 50 C
  • Sensor Supply: 12 VDC(±10%), 30 mA
